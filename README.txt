
# Vikingská oddílová aplikace

## Jak spustit
1. Rozbal ZIP.
2. Otevři soubor `index.html` v prohlížeči.
3. Na mobilu můžeš přes menu prohlížeče zvolit „Přidat na plochu“.

Aplikace obsahuje:
- detailně rozpracované hry,
- pomůckový checklist,
- materiály k tisku,
- CTH plán,
- seznam dívek s poznámkami,
- přiložené vizuály,
- odpočet do konce tábora,
- krizový tahák pro vedoucí.

Poznámky a zaškrtávání se ukládají pouze v daném prohlížeči.
